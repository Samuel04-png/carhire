# Shark Car Hire

Premium car hire website and admin dashboard for Shark Car Hire.

## Local development

Prerequisites:
- Node.js 20+

Run locally:
1. Install dependencies with `npm install`
2. Start the dev server with `npm run dev`
3. Open `http://localhost:3000`

## Scripts

- `npm run dev` starts the Vite dev server on port `3000`
- `npm run build` creates the production build
- `npm run preview` previews the production build locally
- `npm run lint` runs TypeScript checks

## Notes

- No AI provider keys are required for the current build.
- If hot reload is unstable in your local environment, set `DISABLE_HMR="true"` in `.env.local`.
